# Caesar Cipher Encryption & Decryption

## 📌 Project Description
This project is a simple implementation of the **Caesar Cipher Algorithm** using programming logic.  
The program allows users to:

- Encrypt a message
- Decrypt a message
- Enter a custom shift value

The Caesar Cipher is one of the oldest and simplest encryption techniques where each letter in the text is shifted by a fixed number of positions in the alphabet.

---

## 🚀 Features
- Encrypt plain text into cipher text
- Decrypt cipher text back into plain text
- User-defined shift value
- Supports uppercase and lowercase letters
- Simple and beginner-friendly implementation

---

## 🛠️ Technologies Used
- Programming Language: JavaScript

---

## 📖 How Caesar Cipher Works
Example with shift value **3**:

Plain Text:
HELLO

Encrypted Text:
KHOOR

Explanation:
- H → K
- E → H
- L → O
- L → O
- O → R
